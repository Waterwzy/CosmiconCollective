## 游戏逻辑

游戏为回合制游戏，分轮次进行，每轮可以分为以下阶段：

1. 开始阶段

    - 每轮次双方玩家都会交换攻守方，如果为第一轮，随机决定攻守方。

    - 玩家在游戏开始时有默认的骰子配置（由具体卡牌决定，但是游戏中可以变化）

2. 攻击方投掷阶段

    - 攻击方可以投掷骰子，并选定一系列骰子作为本轮的攻击骰子，骰子点数之和为攻击方点数。

    - 在本轮首次抛出后，选定之前，攻击方可以选定任意数量的骰子进行重投，攻击方有2次重投机会（可能受特殊效果影响）。

3. 防御方投掷阶段

    - 防御方可以投掷骰子，并选定一系列骰子作为本轮的防御骰子，骰子点数之和为攻击方点数。

    - 与攻击方不同，防御方没有重投机会（可能受特殊效果影响）。

4. 结算阶段

    - 攻击方点数 - 防御方点数 + 特殊效果影响为本回合攻击方对防御方造成的伤害。

    - 特殊效果对结算的影响可能体现在如下方面：

        - 增加攻击方/防御方点数

        - 免除攻击方的伤害

同时，不同的玩家卡牌具有不同的技能，可以影响对局的各种方面。

效果不同于卡牌技能，可以附加于玩家卡牌上，可以通过玩家技能添加。

除了常规结算发生的伤害，还可以通过如下效果触发非结算类型的伤害：

    - `瞬伤`：在附加效果的瞬间造成一定伤害。

    - `中毒`：在回合结算后，下回合前造成伤害。

    - `反伤`：在结算时攻击方对防御方造成伤害时防御方对攻击方造成伤害。

效果比较特殊，有不可叠加的效果和可叠加的效果。可叠加的效果有“叠加层数”之说，一般来说这个层数决定了效果的能力大小。
