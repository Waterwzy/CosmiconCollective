# 银河战力党

复刻《崩坏：星穹铁道》银河战力党的效果。

目前处于后端构建阶段，未来逐渐完善后会构建前端。

项目结构:

```text
├ core #核心py文件
    └ player #玩家相关
        ├ default.py #默认配置，包含卡牌和效果的配置
        ├ dice.py #骰子基类
        ├ effects.py #效果基类
        └ player.py #玩家卡牌基类
├ main.py #主入口，游戏管理器和效果钩子的定义
└ README.md #本文件
```

目前已完成角色卡牌：

- 火花花

- 阮·梅造物

- 奇美拉

- 开拓妖精

- 大地兽

目前已完成效果：

- 骇入

- 中毒

- 瞬伤

其他未完成内容都在todolist覆盖范围内~

注意：本项目采取 `ruff` 格式化代码，提交 PR 也请提前使用 `ruff` 格式化你的代码。
